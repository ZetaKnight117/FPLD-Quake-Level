# Escape the Terror

This is a First Person level for the game Quake. Everyone in the facility is gone except you. You will have to fight you way though the facilty and kill all the enemies that has taking over the place. Warning you are alone. No one can help you. No one will be able to save you. It's you vs a whole army of enemies to kill.
